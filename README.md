# SC2002-Project

## How to run 
- Required: [gradle](https://gradle.org/install/) installed on the system 
- [Java](https://www.java.com/en/) installed on system 

In `CAMS Application` directory

First Run 

`gradle wrapper`

Subsequent Runs

`.\gradlew run`
